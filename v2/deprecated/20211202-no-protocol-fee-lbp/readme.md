# 2021-12-02 - Liquidity Bootstrapping Pool with no Protocol Fees

> ⚠️ **DEPRECATED: do not use** ⚠️
> This has been deprecated as part of the general migration to V3. See [this X post](https://x.com/Balancer/status/2001695563268387210).
