# 2025-12-05 - V3 Fixed Price Liquidity Bootstrapping Pool

Deployment of the fixed price Liquidity Bootstrapping Pool; a simple kind of LBP that sells a project token at a fixed price. It is seedless - no reserve tokens required - and does not support migration.
